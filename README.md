# iSH-tools
Security toolkit for iSH Shell.

![Image](https://github.com/user-attachments/assets/cd5c653c-e553-4ae8-941a-a7705195edf1) 

## Single-Paste Install: 
```
apk add python3 
apk add py3-pip 
apk add bash
apk add git 
pip3 install pystyle 
git clone https://github.com/cons0le7/iSH-tools
cd ./iSH-tools/tools
chmod +x *.sh
cd ./.. 
python3 main.py
```
